> A => B => EnumerableType<C, D> => EnumerableType<A, B>

Creates a enumerable based on a value and optional key.

``` javascript
of("aaa")("bbb")({}) // {aaa: "bbb"}
of(null)("bbb")([]) // ["bbb"]
```
